阿里云centOS配置Nigux

2019-04-18
  1. centOS自带yum源，所以可以直接使用
  2. 安装
    sudo yum install -y nginx
  3. 默认安装目录： /usr/share/nginx/
  4. 启动
    sudo stystemctl start nginx.service
  5. 开机启动
    sudo systemctl enable nginx.service
  6. 网站文件存放默认目录
    /usr/share/nginx/html
  7. 站默认站点配置
    /etc/nginx/conf.d/default.conf
  8. 自定义Nginx站点配置文件存放目录
    /etc/nginx/conf.d/
  9. Nginx全局配置
    /etc/nginx/nginx.conf
  10. 彻底卸载Nginx（其他软件相同）
  11. 停止Nginx软件
    service nginx stop
  12. 删除Nginx的自动启动
    chkconfig nginx off
  13. 从源头删除Nginx
    rm -rf /usr/sbin/nginx
rm -rf /etc/nginx
rm -rf /etc/init.d/nginx
  14. 再使用yum清理
    yum remove nginx

11.安装软件（例如java）

yum search java|grep jdk

或者： yum list 软件名

yum install java-1.8.0-openjdk
  1. 升级所有的软件包
    yum update
  2. 升级某一个软件包 ，以升级 tomcat 为例
    yum update tomcat
  3. 检查可更新的程序
    yum check-update
  4. 查看所有进程
    netstat -nap
  5. 杀死进程
    kill PID编号
